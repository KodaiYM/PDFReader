# object_pool
## 概要
| クラス名 | 概要 |
| --- | --- |
| pdfparser::object_pool | 間接オブジェクトを保持し、間接オブジェクトへのアクセスを管理する |

## 使用例
1. add_xref_table を使ってクロスリファレンステーブルの情報を登録する
2. dereference を使って間接参照情報と、登録されているクロスリファレンステーブルの情報から間接オブジェクトを取得する
