# tokenizer
## 概要
| クラス名 | 概要 |
| --- | --- |
| pdfparser::tokenizer | ストリームのトークン単位の解析を行う |

## 用語
### 文字
| 用語 | 意味 |
| ---- | ---- |
| 空白類文字 | null(00), 水平タブ(09), 改行(0A), 改ページ(0C), 行頭復帰(0D), 20(空白) |
| 区切り文字 | (, ), <, >, [, ], {, }, /, % |
| 標準文字 | その他の文字 |
